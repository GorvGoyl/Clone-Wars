## Contribution

2 ways to add your/their repo to the list:

_(Note: project must have at least minimal functionality, please do not submit any 'UI only' clone)_

1. Add your contribution to [readme.md](https://github.com/Github/Clone-Wars/blob/main/README.md) (_you may use online md editor like [markdown.site](https://markdown.site/) for better table visualization_) and send me a PR! (_faster approval and your name would be added to project contributors_)  
   OR
2. Create a [new issue](https://github.com/Github/Clone-Wars/issues/new) with following details: clone name, code repo link, demo link, and tech stack used.

Any other feedback is also welcome :)
